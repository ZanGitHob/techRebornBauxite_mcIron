# Tech Reborn Bauxite to Iron Ore Conversion Datapack

A lightweight Minecraft: Java Edition Released 1.20.1 Datapack that lets you craft useless piles of Bauxite Ore into valuable Vanilla Iron Ore using simple, shapeless inventory recipes (befitted for some modpacks, not recommended if gated progression is intended as a feature in that modpack, use at your own discretion).

## Why Do You Want to Make This? Bauxite is Needed for Tech Reborn Progressions!
When I was playing latest Minecraft modpacks that has Tech Reborn, I always end up stockpiling a frick ton of Bauxite Ore Blocks (whether Stone or Deepslate). So I made this conversion recipe Datapack to solve this hassle. No more useless Bauxites galore!

---

## Dependencies

Of course, for this Datapack to work, you need **Tech Reborn** in the `mods` folder.

---

## What's inside this Pack?

Both crafting recipes are **2x2 shapeless**.

* **Stone Bauxite Ore** to **Minecraft Iron Ore**.
* **Deepslate Bauxite Ore** to **Minecraft Deepslate Iron Ore**.

In the backend, more specifically, `techreborn:bauxite_ore` and `techreborn:deepslate_bauxite_ore` are crafted into `minecraft:iron_ore` and `minecraft:deepskate_iron_ore`. 

---

## How to Install?

1. Download the `.zip` file.
2. Boot Minecraft.
3. Choose a world, click **Edit**, and then click **Open World Folder**.
4. Open the `datapacks` folder.
5. Unzip this folder (sometimes zipped folder are not detected by the game) and drop it into the world folder.
6. Hop right into the world, ta-dah!
7. If you are already inside the world, put `/reload` in the chat to refresh the recipes.
